User Guide

Editing the network

Adding a node to the network

1. On the toolbar on the left, click on the type of node to draw.
2. The desired block will appear on the blank space with default and valid values

Editing a node in the network

1. Do one of the following operations:
1.1 By right-clicking on a selected node
1.2 Or selecting a node and using the shortcut Ctrl+E
1.3 Or selecting a node and going on Edit > Edit on the menu bar
2. A dialog will appear with all parameters of that node.
3. Some parameter will not be editable, such as weight and bias attributes which depend on the input, or the input itself if there is a node above the one to change.
4. Type the new values of the editable parameters.
5. Changes will be applied pushing "Apply" button, and they will be visible on the scene.
If an error occurs, the node will not be changed, and a specific message will appear.
Pressing "Cancel" or closing the dialog instead of pressing "Apply", the node will not change.

Copying a node

1. Select one or more (with Ctrl key) nodes to copy.
2. Copy the node by:
1.1 Using the shortcut Ctrl+C
1.2 Right-clicking on one of the selected nodes and clicking on "Copy"
1.3 Selecting Edit > Copy on the menu bar
3. The selected items will be copied to clipboard.

Pasting a node

1. Paste one or more copied nodes by:
1.1 Using the shortcut Ctrl+V
1.2 Selecting Edit > Paste on the menu bar

Cutting a node

1. Select one or more (with Ctrl key) nodes to cut.
2. Cur the node by:
1.1 Using the shortcut Ctrl+X
1.2 Right-clicking on one of the selected nodes and clicking on "Cut"
1.3 Selecting Edit > Cut on the menu bar
3. The selected items will be deleted from the scene and copied to clipboard.
Alert: all the connections of the deleted item will be deleted, and if it was on the middle of the network, a new connection will be automatically drawn substituting the deleted node (if logically possible). If a new connection is not legal for the network, it will be split into two networks.

Deleting a node

1. Select one or more (with Ctrl key) nodes to delete.
2. Delete the node by:
1.1 Using the shortcut Del
1.2 Right-clicking on one of the selected nodes and clicking on "Delete"
1.3 Selecting Edit > Delete on the menu bar
3. The selected items will be deleted from the scene. Alert: all the connections of the deleted item will be deleted, and if it was on the middle of the network, a new connection will be automatically drawn substituting the deleted node (if logically possible). If a new connection is not legal for the network, it will be split into two networks.

Drawing a connection

1. Enter in the drawing mode by:
1.1 Clicking on the "Draw Line" button on the toolbar on the left
1.2 Using the shortcut Ctrl+D
1.3 Selecting Edit > Draw Connection on the menu bar
2. Click on the first node of the connection: clicking on a node which already have a connection starting from it will cause an error.
3. Click on the second node of the connection: clicking on a node which have a connection going into it will cause an error.
4. A line will be drawn between the two nodes (showing the output of the first node) if the logical connection between them is legal, otherwise the connection will not be drawn, and a specific message will appear

Deleting a connection

1. Click on the connection do delete
2. Delete the connection by:
2.1 Using the shortcut Del or Ctrl+X
2.2 Selecting Edit > Delete or Edit > Cut on the menu bar
3. The connection will be deleted from the network Alert: if deleting a connection involves splitting the network in two networks, it will not be possible to save the network until it becomes again a connected sequential network.

Insert a node between two connected nodes

1. Enter in the drawing mode by:
1.1 Clicking on the "Insert block" button on the toolbar on the left
1.2 Using the shortcut Ctrl+I
1.3 Selecting Edit > Insert Block in Connection on the menu bar
2. Click the node to insert in a connection or in the connection itself
3. Click on the connection or in the node to insert in it
4. If the new connections are legal, the clicked
connection will disappear, and two new connections will be drawn to insert the clicked block in the network.

Clearing the network

1. Clear the network by:
1.1 Using the shortcut Ctrl+Shift+C
1.2 Selecting Edit > Clear on the menu bar
2. All nodes and connection will be deleted from the network.

Editing the interface

Zoom in and Zoom out

1. Zoom in/out by:
1.1 Using the shortcut Ctrl++/Ctrl+-
1.2 Selecting View > Zoom in/Zoom out
2. The blocks in the scene are bigger/smaller of a 1.5 factor

Show and hide toolbars

1. Show or hide Tools bar or Nodes bas selecting View > Show > Tools/Blocks Library

Show node properties

1. Select the node whose details are required
2. Complete the operation by:
2.1 Using the shortcut Ctrl+P
2.2 Selecting View > Properties
2.3 Right clicking on the node and clicking "Properties"
3. A toolbar on the right of the window will appear with the complete description of the node type.
